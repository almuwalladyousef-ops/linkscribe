# LinkScribe — Intake

- **What:** A web app where Yousef pastes a video link from YouTube, Instagram, TikTok, or another platform and gets the transcript.
- **Purpose:** Personal
- **Type:** Web app first, desktop app later
- **For:** Just Yousef
- **MVP:** Paste video link → click transcribe → see transcript → copy transcript.
- **Backend needed:** Yes
- **Database needed:** No for MVP
- **Stack:** Next.js web app, backend API route, `yt-dlp` for supported video/audio extraction, Whisper transcription locally or API-based during planning.
